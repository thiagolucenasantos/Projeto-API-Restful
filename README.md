# Projeto-API-Restful
API Restful de produtos seguindo modelo de  maturidade de Leonard Richardson.

Para atender os padrões de uma API Restful foi utilizado.
- Spring Boot 3 E Java 17.
- Spring Data JPA
- Spring Web MVC
- Spring Validation
- Spring Hateos
